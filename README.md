# FitHelper

A lightweight, intuitive health utility for macOS — providing quick, easy-to-understand functions that help people save time and live healthier.

## Prerequisites

- Node.js 22+
- macOS

## Setup

```bash
npm install
```

Place your OpenAI API key in `keys/open-ai-api-key.txt` to enable the AI Coach feature.

## Development

```bash
npm run dev
```

## Testing

```bash
npm test
```

## Build

```bash
npm run build
```

Produces a `.dmg` installer in the `release/` directory.

## Tools

1. **Pace Converter** — Bidirectional conversion between mph and min/km for runners.
2. **Calorie Library** — Editable categorized food calorie reference with preset data.
3. **Daily Calorie Tracker** — Daily intake tracking with drag-reorder, eaten/planned toggle, and smart food suggestions.
4. **Training Log + AI Coach** — Freeform training records and plan with LLM-powered coaching analysis via OpenAI.
